Try to solve sentiment analysis problem

盒子在日本
2020-06-04
Daily

就一整天在做movie review分类

一开始的试行错误是用单词index
果然无论怎么训练，预测率都是在50%左右
后来换成了bag-of-word
输入的dimension是2000
只有一个hidden层，10个神经元
最终结构是
2000 -> 10 -> relu -> 4 -> softmax -> cross entropy
test数据的成功率就到达了88%左右
